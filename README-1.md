# 🚀 STP Playwright Starter Pack Template

This folder contains the **Playwright Starter Pack**.

👉 This is the part you use to create a new test automation project.

---

## 🧭 How to use

1. Copy or initialise this template into your project
2. Run setup:

   ```bash
   ./scripts/init-starter-pack.sh
   ```
3. Run tests

    ```bash
    npm test
    ```

## 📦 What’s included
- Playwright test framework (src/)

- CI/CD pipelines (pipelines/)

- Setup scripts (scripts/)

- Validation test (validation/)
