# 🚀 Playwright App Starter Pack

## What is this?

This folder contains the **STP Playwright Starter Pack template**.

👉 This is the part you copy/use when creating a new test automation project.

---

## What do I do with this?

1. Copy or initialise this template into your project
2. Run the setup script (`scripts/init-starter-pack.sh`)
3. Execute tests using the provided pipeline or locally

---

## What is NOT part of this?

The parent repository contains:
- platform configuration
- reference implementations
- documentation

👉 You only need the contents of this template to get started.
