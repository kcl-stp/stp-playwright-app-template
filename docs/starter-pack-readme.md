# 🚀 STP Playwright App Starter Pack

## What is this?

This repository is the **STP Playwright App Starter Pack**.

It provides a reusable baseline for creating a Playwright-based UI test automation project with agreed STP structure, documentation, and pipeline patterns.

---

## What is included?

This starter pack includes baseline examples for:

- Playwright test structure
- sample tests
- configuration
- pipeline integration
- documentation and runbook guidance
- reporting patterns

---

## How do I use it?

1. Create or initialise your new test automation repository from this starter pack
2. Review and update the project-specific configuration
3. Complete the setup runbook and checklist
4. Run the tests locally
5. Run the pipeline validation flow

---

## What should teams customise?

Teams should update this starter pack for their own application needs, for example:

- environment configuration
- credentials and token handling
- test data preparation
- tagging strategy
- application-specific page objects and tests
- project-specific pipeline variables

See the docs folder for setup, pipelines, and running tests guidance.

---

## What stays standard?

The aim is to keep the following aligned with STP standards where possible:

- repository structure
- baseline pipeline approach
- SDLC / PR validation pattern
- reporting approach
- documentation structure
